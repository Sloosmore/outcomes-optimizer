# runtime/skills/cloudflare-browser

Skill for controlling headless Chrome via Cloudflare Browser Rendering using the Chrome DevTools Protocol (CDP) over WebSocket. Used for screenshots, page navigation, scraping, and video capture in Cloudflare Workers environments.

See `SKILL.md` for full usage documentation and CDP command reference.
