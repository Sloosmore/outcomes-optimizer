# skills/agent-email

Version-controlled skill that teaches agents how to manage temporary email inboxes using the `agent-email` CLI. Covers inbox initialization, email waiting, reading, and link/code extraction across three adapter backends (gmail, mail.tm, 1secmail).

See `SKILL.md` for the complete skill specification, adapter comparison table, and usage documentation.
