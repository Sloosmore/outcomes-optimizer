# skills/agent-media

Version-controlled skill that teaches agents how to generate media (images, video, audio) using the `agent-media` CLI. Covers DALL-E image generation, Veo video generation, TTS audio, captioning, assembly, and video post-processing.

See `SKILL.md` for the complete skill specification, cost safety guidance, and usage examples.
