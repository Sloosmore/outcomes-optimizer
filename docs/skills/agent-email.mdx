## Purpose

Provides procedures for managing disposable email inboxes during agent workflows — account creation, verification link extraction, and email monitoring.

## Backed By

The `throwmail` CLI package. See [throwmail package](/docs/packages/agent-email) for CLI reference.

## Common Use Cases

- Create a disposable inbox for account signup
- Wait for a verification email
- Extract and follow a verification link
